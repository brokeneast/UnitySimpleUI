# Unity Simple UI
平時Unity專案會使用到的一些遊戲系統UI範例實現。

### 使用方式
----
#### 初始化
1. 於Hierarchy右鍵增添Canvas。
2. 於Hierarchy右鍵新增UIManager。
3. 於UIManager中Orientation中選擇遊戲為直向(Portrait)、橫向(Landscape)或是自行判斷方向(Auto)。

### 介面管理員
----
負責調用系統性UI。
#### 彈跳視窗 Dialog
提醒視窗作為使用者動作確認或提示，最多為兩種選擇，通常為確認即取消，適合運用在系統提示、警告等。
選擇視窗為多選一，可客制化按鍵內容及操作。
資訊視窗適合運用於大量文字資訊，其內容將附有滾動條。例如:隱私權聲明視窗等。
- [x] 提醒
- [x] 選擇
- [x] 資訊

#### 讀取介面 Loading
讀取視窗視用於異步情形。提示型將有讀取介面。
- [ ] 透明
- [ ] 提示
- [ ] 讀取條
- [ ] 環形讀取

#### 提示介面 Toast
適用於簡單的提示通知，不影響使用者體驗為原則，將於一段時間後自動消逝淡出。
- [x] 一般
- [x] 消逝
